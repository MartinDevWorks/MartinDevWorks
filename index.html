Sobre Mí

Diseño sitios web atractivos, funcionales y personalizados, combinando creatividad y tecnología para garantizar una experiencia de usuario excepcional. Desarrollo aplicaciones web rápidas, escalables y compatibles con todas las plataformas, optimizadas para brindar una experiencia fluida en Android, iOS y navegadores web.

Más sobre mí

Desde que inicié mi camino como Desarrollador front-end en 2023, he adquirido un sólido dominio de las tecnologías fundamentales para la creación de experiencias web modernas. Mi aprendizaje se centra en desarrollar interfaces responsivas, accesibles y optimizadas, utilizando HTML5 semántico para estructuración robusta, CSS3 con técnicas avanzadas (Flexbox, Grid, animaciones) para diseños adaptables, y JavaScript ES6+ para interactividad dinámica. Complemento estas bases con herramientas como Git para control de versiones, metodología mobile-first, y principios de rendimiento web (optimización de assets, carga progresiva). Actualmente estoy expandiendo mis habilidades hacia React.js y TypeScript para construir aplicaciones escalables.
